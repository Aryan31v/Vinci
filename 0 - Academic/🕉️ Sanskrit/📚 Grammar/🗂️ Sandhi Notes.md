## ЁЯФ╣ Ach Sandhi (рд╕реНрд╡рд░ рд╕рдиреНрдзрд┐)
*Sutras involving the transformation of vowels.*

1. **рдпрдгреН рд╕рдиреНрдзрд┐рдГ (рдЗрдХреЛ рдпрдгрдЪрд┐):**
   - *Rule:* рдЗ/рдИ/рдЙ/рдК/рдЛ + рдЕрд╕рдорд╛рди рд╕реНрд╡рд░ = рдпреН/рд╡реН/рд░реН
   - *Example:* рдЕрддрд┐ + рдЕрдиреНрддрдГ = рдЕрддреНрдпрдиреНрддрдГ
2. **рдЕрдпрд╛рджрд┐ рд╕рдиреНрдзрд┐рдГ (рдПрдЪреЛрд╜рдпрд╡рд╛рдпрд╛рд╡рдГ):**
   - *Rule:* рдП/рдР/рдУ/рдФ + рдХреЛрдИ рдЕрдиреНрдп рд╕реНрд╡рд░ = рдЕрдпреН/рдЖрдпреН/рдЕрд╡реН/рдЖрд╡реН
   - *Example:* рдиреЗ + рдЕрдирдореН = рдирдпрдирдореН
3. **рд╡рд╛рдиреНрддреЛ рдпрд┐ рдкреНрд░рддреНрдпрдпреЗ:**
   - *Rule:* рдУ/рдФ + 'рдп' рд╕реЗ рд╢реБрд░реВ рд╣реЛрдиреЗ рд╡рд╛рд▓рд╛ рдкреНрд░рддреНрдпрдп = рдЕрд╡реН/рдЖрд╡реН
4. **рдПрдЩрд┐ рдкрд░рд░реВрдкрдореН:**
   - *Rule:* рдЕ-рдХрд╛рд░рд╛рдиреНрдд рдЙрдкрд╕рд░реНрдЧ + рдП/рдУ рд╕реЗ рд╢реБрд░реВ рд╣реЛрдиреЗ рд╡рд╛рд▓реА рдзрд╛рддреБ = рдкрд░рд░реВрдк (The last 'a' of prefix merges).
5. **рд╕рд╡рд░реНрдгрджреАрд░реНрдШ рд╕рдиреНрдзрд┐рдГ (рдЕрдХрдГ рд╕рд╡рд░реНрдгреЗ рджреАрд░реНрдШрдГ):**
   - *Rule:* рдЕ/рдЖ + рдЕ/рдЖ = рдЖ (Similar vowels merge into long).
   - *Example:* рд╡рд┐рджреНрдпрд╛ + рдЖрд▓рдпрдГ = рд╡рд┐рджреНрдпрд╛рд▓рдпрдГ
6. **рдкреВрд░реНрд╡рд░реВрдк рд╕рдиреНрдзрд┐рдГ (рдПрдЩрдГ рдкрджрд╛рдиреНрддрд╛рджрддрд┐):**
   - *Rule:* рдП/рдУ (at end of word) + рдЕ = рдПрд╜ / рдУрд╜ (The 'a' elides and leaves an Avagraha).

## ЁЯФ╣ Hal Sandhi (рд╡реНрдпрдЮреНрдЬрди рд╕рдиреНрдзрд┐)
*Sutras involving the transformation of consonants.*

7. **рд╢реНрдЪреБрддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рд╕реНрддреЛрдГ рд╢реНрдЪреБрдирд╛ рд╢реНрдЪреБрдГ):**
   - *Rule:* рд╕реН/рддреН рд╡рд░реНрдЧ + рд╢реН/рдЪреН рд╡рд░реНрдЧ = рд╢реН/рдЪреН рд╡рд░реНрдЧ
   - *Example:* рд░рд╛рдордГ + рдЪ = рд░рд╛рдорд╢реНрдЪ
8. **рд╖реНрдЯреБрддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рд╖реНрдЯреБрдирд╛ рд╖реНрдЯреБрдГ):**
   - *Rule:* рд╕реН/рддреН рд╡рд░реНрдЧ + рд╖реН/рдЯреН рд╡рд░реНрдЧ = рд╖реН/рдЯреН рд╡рд░реНрдЧ
   - *Example:* рдХреГрд╖реН + рдирдГ = рдХреГрд╖реНрдгрдГ
9. **рд▓рддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рддреЛрд░реНрд▓рд┐):**
   - *Rule:* рддреН рд╡рд░реНрдЧ + рд▓ = рд▓реНрд▓реН
   - *Example:* рдЙрддреН + рд▓реЗрдЦрдГ = рдЙрд▓реНрд▓реЗрдЦрдГ
10. **рдЬрд╢реНрддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рдЭрд▓рд╛рдВ рдЬрд╢реЛрд╜рдиреНрддреЗ):**
    - *Rule:* рд╡рд░реНрдЧ рдХрд╛ рдкреНрд░рдердо рд╡рд░реНрдг тЮФ рддреГрддреАрдп рд╡рд░реНрдг (At the end of a word).
    - *Example:* рд╡рд╛рдХреН + рдИрд╢рдГ = рд╡рд╛рдЧреАрд╢рдГ
11. **рдЭрдпреЛ рд╣реЛрд╜рдиреНрдпрддрд░рд╕реНрдпрд╛рдореН:**
    - *Rule:* рдЭрдпреН + рд╣ = Optionally 'h' turns into 4th letter of previous class.
12. **рд╢рд╢реНрдЫреЛрд╜рдЯрд┐:**
    - *Rule:* рддреН + рд╢ = рдЪреНрдЫ
    - *Example:* рддрддреН + рд╢рд┐рд╡рдГ = рддрдЪреНрдЫрд┐рд╡рдГ
13. **рдЕрдиреБрдирд╛рд╕рд┐рдХ рд╕рдиреНрдзрд┐рдГ (рдпрд░реЛрд╜рдиреБрдирд╛рд╕рд┐рдХреЗрд╜рдиреБрдирд╛рд╕рд┐рдХреЛ рд╡рд╛):**
    - *Rule:* рд╡рд░реНрдЧ рдХрд╛ рдкреНрд░рдердо рд╡рд░реНрдг тЮФ рдкрдЮреНрдЪрдо рд╡рд░реНрдг
14. **рдЕрдиреБрд╕реНрд╡рд╛рд░ рд╕рдиреНрдзрд┐рдГ (рдореЛрд╜рдиреБрд╕реНрд╡рд╛рд░рдГ):**
    - *Rule:* рдореН + рд╡реНрдпрдЮреНрдЬрди = рдЕрдиреБрд╕реНрд╡рд╛рд░ (рдВ)
15. **рдирд╢реНрдЪрд╛рдкрджрд╛рдиреНрддрд╕реНрдп рдЭрд▓рд┐:**
    - *Rule:* рдиреН/рдореН (not at end) + рдЭрд▓реН = рдЕрдиреБрд╕реНрд╡рд╛рд░
16. **рддреБрдХ-рдЖрдЧрдо (рдЫреЗ рдЪ):**
    - *Rule:* рд╣реНрд░рд╕реНрд╡ рд╕реНрд╡рд░ + рдЫ = рдЪреНрдЫ
17. **рдЪрд░реНрддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рдЦрд░рд┐ рдЪ):**
    - *Rule:* рдЭрд▓реН + рдЦрд░реН = рдкреНрд░рдердо рд╡рд░реНрдг (3rd/4th turn into 1st).
18. **рдкрд░рд╕рд╡рд░реНрдг рд╕рдиреНрдзрд┐рдГ (рд╡рд╛ рдкрджрд╛рдиреНрддрд╕реНрдп):**
    - *Rule:* рдЕрдиреБрд╕реНрд╡рд╛рд░ + рд╡реНрдпрдЮреНрдЬрди = Next letter's class 5th letter.
