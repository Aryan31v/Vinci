## ЁЯФ╣ 1. Ach Sandhi (рд╕реНрд╡рд░ рд╕рдиреНрдзрд┐)
*Sutras involving the transformation of vowels.*

### 1.1 рдпрдгреН рд╕рдиреНрдзрд┐рдГ (рдЗрдХреЛ рдпрдгрдЪрд┐)
- **Sutra:** `рдЗрдХреЛ рдпрдгрдЪрд┐` (6.1.77)
- **Rule:** When **рдЗ/рдИ, рдЙ/рдК, рдЛ/реа, рдМ** are followed by a **dissimilar vowel** (рдЕрдЪреН), they are replaced by **рдпреН, рд╡реН, рд░реН, рд▓реН** respectively.
- **Formula:** `Ik (рдЗ, рдЙ, рдЛ, рдМ) + Ach (Vowel) = Yan (рдпреН, рд╡реН, рд░реН, рд▓реН)`
- **Examples:**
    - рдЕрддрд┐ + рдЕрдиреНрддрдГ = **рдЕрддреНрдпрдиреНрддрдГ** (i + a = y)
    - рд╡рд┐ + рдЖрдпрд╛рдо = **рд╡реНрдпрд╛рдпрд╛рдордГ** (i + ─Б = y─Б) *[From Chaos Stream]*
    - рд╕реБ + рдЖрдЧрддрдореН = **рд╕реНрд╡рд╛рдЧрддрдореН** (u + ─Б = v─Б)
    - рдкрд┐рддреГ + рдЖрджреЗрд╢рдГ = **рдкрд┐рддреНрд░рд╛рджреЗрд╢рдГ** (с╣Ы + ─Б = r─Б)

### 1.2 рдЕрдпрд╛рджрд┐ рд╕рдиреНрдзрд┐рдГ (рдПрдЪреЛрд╜рдпрд╡рд╛рдпрд╛рд╡рдГ)
- **Sutra:** `рдПрдЪреЛрд╜рдпрд╡рд╛рдпрд╛рд╡рдГ` (6.1.78)
- **Rule:** When **рдП, рдУ, рдР, рдФ** are followed by **any vowel**, they become **рдЕрдпреН, рдЕрд╡реН, рдЖрдпреН, рдЖрд╡реН** respectively.
- **Formula:** `Ech (рдП, рдУ, рдР, рдФ) + Ach = Ay, Av, ─Аy, ─Аv`
- **Examples:**
    - рдиреЗ + рдЕрдирдореН = **рдирдпрдирдореН** (e + a = ay)
    - рдкреЛ + рдЕрдирдГ = **рдкрд╡рдирдГ** (o + a = av)
    - рдиреИ + рдЕрдХрдГ = **рдирд╛рдпрдХрдГ** (ai + a = ─Бy)
    - рдкреМ + рдЕрдХрдГ = **рдкрд╛рд╡рдХрдГ** (au + a = ─Бv)

### 1.3 рдЧреБрдг рд╕рдиреНрдзрд┐рдГ (рдЖрджреНрдЧреБрдгрдГ)
- **Sutra:** `рдЖрджреНрдЧреБрдгрдГ` (6.1.87)
- **Rule:** рдЕ/рдЖ + рдЗ/рдИ = **рдП** | рдЕ/рдЖ + рдЙ/рдК = **рдУ** | рдЕ/рдЖ + рдЛ = **рдЕрд░реН**
- **Examples:**
    - рджреЗрд╡ + рдЗрдиреНрджреНрд░рдГ = **рджреЗрд╡реЗрдиреНрджреНрд░рдГ**
    - рд╕реВрд░реНрдп + рдЙрджрдпрдГ = **рд╕реВрд░реНрдпреЛрджрдпрдГ**

### 1.4 рд╡реГрджреНрдзрд┐ рд╕рдиреНрдзрд┐рдГ (рд╡реГрджреНрдзрд┐рд░реЗрдЪрд┐)
- **Sutra:** `рд╡реГрджреНрдзрд┐рд░реЗрдЪрд┐` (6.1.88)
- **Rule:** рдЕ/рдЖ + рдП/рдР = **рдР** | рдЕ/рдЖ + рдУ/рдФ = **рдФ**
- **Examples:**
    - рдПрдХ + рдПрдХрдореН = **рдПрдХреИрдХрдореН**
    - рд╡рди + рдУрд╖рдзрд┐рдГ = **рд╡рдиреМрд╖рдзрд┐рдГ**

### 1.5 рд╕рд╡рд░реНрдгрджреАрд░реНрдШ рд╕рдиреНрдзрд┐рдГ (рдЕрдХрдГ рд╕рд╡рд░реНрдгреЗ рджреАрд░реНрдШрдГ)
- **Sutra:** `рдЕрдХрдГ рд╕рд╡рд░реНрдгреЗ рджреАрд░реНрдШрдГ` (6.1.101)
- **Rule:** Two similar vowels (Short or Long) merge into their **Long** form.
- **Examples:**
    - рд╡рд┐рджреНрдпрд╛ + рдЖрд▓рдпрдГ = **рд╡рд┐рджреНрдпрд╛рд▓рдпрдГ**
    - рдХрдкреА + рдИрд╢рдГ = **рдХрдкреАрд╢рдГ**

### 1.6 рдкреВрд░реНрд╡рд░реВрдк рд╕рдиреНрдзрд┐рдГ (рдПрдЩрдГ рдкрджрд╛рдиреНрддрд╛рджрддрд┐)
- **Sutra:** `рдПрдЩрдГ рдкрджрд╛рдиреНрддрд╛рджрддрд┐` (6.1.109)
- **Rule:** If **рдП/рдУ** at the end of a word is followed by short **рдЕ**, the 'рдЕ' merges into the previous vowel and is indicated by an Avagraha (**рд╜**).
- **Examples:**
    - рд╣рд░реЗ + рдЕрддреНрд░ = **рд╣рд░реЗрд╜рддреНрд░**
    - рд╡рд┐рд╖реНрдгреЛ + рдЕрд╡ = **рд╡рд┐рд╖реНрдгреЛрд╜рд╡**

### 1.7 рдкрд░рд░реВрдк рд╕рдиреНрдзрд┐рдГ (рдПрдЩрд┐ рдкрд░рд░реВрдкрдореН)
- **Sutra:** `рдПрдЩрд┐ рдкрд░рд░реВрдкрдореН` (6.1.94)
- **Rule:** Preposition ending in 'a' (рдЕ-рдХрд╛рд░рд╛рдиреНрдд рдЙрдкрд╕рд░реНрдЧ) + Verb starting with e/o = The prefix vowel merges into the verb vowel.
- **Examples:**
    - рдкреНрд░ + рдПрдЬрддреЗ = **рдкреНрд░реЗрдЬрддреЗ**

## ЁЯФ╣ 2. Hal Sandhi (рд╡реНрдпрдЮреНрдЬрди рд╕рдиреНрдзрд┐)
*Sutras involving the transformation of consonants.*

### 2.1 рд╢реНрдЪреБрддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рд╕реНрддреЛрдГ рд╢реНрдЪреБрдирд╛ рд╢реНрдЪреБрдГ)
- **Rule:** `S/T-Group` (рд╕реН/рддреН/рдереН/рджреН/рдзреН/рдиреН) + `┼Ъ/Ch-Group` (рд╢реН/рдЪреН/рдЫреН/рдЬреН/рдЭреН/рдЮреН) = **┼Ъ/Ch-Group**.
- **Logic:** Dental becomes Palatal when in contact with Palatal.
- **Examples:**
    - рд░рд╛рдордГ + рдЪ = рд░рд╛рдорд╕реН + рдЪ = **рд░рд╛рдорд╢реНрдЪ**
    - рд╕рддреН + рдЪрд┐рддреН = **рд╕рдЪреНрдЪрд┐рддреН**

### 2.2 рд╖реНрдЯреБрддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рд╖реНрдЯреБрдирд╛ рд╖реНрдЯреБрдГ)
- **Rule:** `S/T-Group` + `с╣в/с╣м-Group` (рд╖реН/рдЯреН/рдареН/рдбреН/рдвреН/рдгреН) = **с╣в/с╣м-Group**.
- **Logic:** Dental becomes Retroflex when in contact with Retroflex.
- **Examples:**
    - рд░рд╛рдорд╕реН + рд╖рд╖реНрдардГ = **рд░рд╛рдорд╖реНрд╖рд╖реНрдардГ**
    - рддрддреН + рдЯреАрдХрд╛ = **рддрдЯреНрдЯреАрдХрд╛**
    - рдХреГрд╖реН + рдирдГ = **рдХреГрд╖реНрдгрдГ** (Here 'n' becomes 'с╣З' due to contact with 'с╣г')
    - рддреГрд╖реН + рдХреНрд╡рд┐рдкреН = **рддреГрдЯреН** (Special Derivation: Padanta 'с╣г' becomes 'с╣н') *[From Chaos Stream]*

### 2.3 рдЬрд╢реНрддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рдЭрд▓рд╛рдВ рдЬрд╢реЛрд╜рдиреНрддреЗ)
- **Rule:** At the end of a word (Pada), any hard consonant (Jhal) becomes the **3rd letter** of its class (Jash).
- **Examples:**
    - рд╡рд╛рдХреН + рдИрд╢рдГ = **рд╡рд╛рдЧреАрд╢рдГ** (k -> g)
    - рдЬрдЧрддреН + рдИрд╢рдГ = **рдЬрдЧрджреАрд╢рдГ** (t -> d)

### 2.4 рдЕрдиреБрд╕реНрд╡рд╛рд░ рд╕рдиреНрдзрд┐рдГ (рдореЛрд╜рдиреБрд╕реНрд╡рд╛рд░рдГ)
- **Rule:** `рдореН` (at end of word) + Any Consonant = **Anusvara (рдВ)**.
- **Examples:**
    - рд╣рд░рд┐рдореН + рд╡рдиреНрджреЗ = **рд╣рд░рд┐рдВрд╡рдиреНрджреЗ**
    - рдЧрдЬрдореН + рд╕рд┐рдВрд╣рдореН = **рдЧрдЬрдВ рд╕рд┐рдВрд╣рдореН** *[From Chaos Stream]*
    - **Note:** If followed by a vowel, 'm' remains 'm' (e.g., рдЕрд╣рдореН + рдЕрдкрд┐ = рдЕрд╣рдордкрд┐).

### 2.5 рдкрд░рд╕рд╡рд░реНрдг рд╕рдиреНрдзрд┐рдГ (рдЕрдиреБрд╕реНрд╡рд╛рд░рд╕реНрдп рдпрдпрд┐ рдкрд░рд╕рд╡рд░реНрдгрдГ)
- **Rule:** Anusvara + Yayi (Consonant) = Optional change to the **5th letter** of the following class.
- **Examples:**
    - рдЧрдЩреНрдЧрд╛ (Gam + ga)
    - рд╢рд╛рдиреНрддрдГ (Sham + tah)

### 2.6 рд▓рддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рддреЛрд░реНрд▓рд┐)
- **Rule:** `рддреН` class + `рд▓` = **рд▓реНрд▓реН**.
- **Examples:**
    - рдЙрддреН + рд▓реЗрдЦрдГ = **рдЙрд▓реНрд▓реЗрдЦрдГ**
    - рддрддреН + рд▓реАрдирдГ = **рддрд▓реНрд▓реАрдирдГ**

### 2.7 рдЫрддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рд╢рд╢реНрдЫреЛрд╜рдЯрд┐)
- **Rule:** `рддреН` + `рд╢` = **рдЪреНрдЪ** + **рдЫ** = **рдЪреНрдЫ** (T becomes Ch, ┼Ъ becomes Ch).
- **Examples:**
    - рддрддреН + рд╢рд┐рд╡рдГ = **рддрдЪреНрдЫрд┐рд╡рдГ**

## ЁЯФ╣ 3. Visarga Sandhi (рд╡рд┐рд╕рд░реНрдЧ рд╕рдиреНрдзрд┐)
*Transformation of the Visarga (рдГ).*

### 3.1 рд╕рддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рд╡рд┐рд╕рд░реНрдЬрдиреАрдпрд╕реНрдп рд╕рдГ)
- **Sutra:** `рд╡рд┐рд╕рд░реНрдЬрдиреАрдпрд╕реНрдп рд╕рдГ` (8.3.34)
- **Rule:** Visarga (`рдГ`) + Khar (Hard Consonants: 1st/2nd letters, ┼Ы, с╣г, s) = **рд╕реН** (s).
    - Note: The `s` may further change to `┼Ы` (Schutva) or `с╣г` (Shtutva) depending on the following letter.
- **Examples:**
    - рдирдордГ + рддреЗ = **рдирдорд╕реНрддреЗ**
    - рдЬреНрд╡рд░рдГ + рдЫрд░реНрджрд┐рдГ = рдЬреНрд╡рд░рд╕реН + рдЫрд░реНрджрд┐рдГ = **рдЬреНрд╡рд░рд╢реНрдЫрд░реНрджрд┐рдГ** (Visarga -> s -> ┼Ы due to 'ch') *[From Chaos Stream]*

### 3.2 рдЙрддреНрд╡ рд╕рдиреНрдзрд┐рдГ
**Type A (Before 'a'):** `рдЕрддреЛ рд░реЛрд░рдкреНрд▓реБрддрд╛рджрдкреНрд▓реБрддреЗ` (6.1.113)
- **Rule:** `рдЕрдГ` + `рдЕ` = **рдУ** + **рд╜** (Utva + Purvarupa).
- **Example:** рд╢рд┐рд╡рдГ + рдЕрд░реНрдЪреНрдпрдГ = **рд╢рд┐рд╡реЛрд╜рд░реНрдЪреНрдпрдГ**

**Type B (Before Soft Consonants):** `рд╣рд╢рд┐ рдЪ` (6.1.114)
- **Rule:** `рдЕрдГ` + Hash (Soft Consonants: 3rd, 4th, 5th, h, y, v, r, l) = **рдУ**.
- **Example:** рд╢рд┐рд╡рдГ + рд╡рдиреНрджреНрдпрдГ = **рд╢рд┐рд╡реЛрд╡рдиреНрджреНрдпрдГ**

### 3.3 рд░реБрддреНрд╡ рд╕рдиреНрдзрд┐рдГ (рд╕рд╕рдЬреБрд╖реЛ рд░реБрдГ)
- **Sutra:** `рд╕рд╕рдЬреБрд╖реЛ рд░реБрдГ` (8.2.66)
- **Rule:** Padanta `s` or `Sajush` word = **Ru (рд░реН)**.
    - Used when Visarga is preceded by vowel other than 'a/─Б' and followed by vowel/soft consonant.
- **Example:**
    - рдореБрдирд┐рдГ + рдЕрдпрдореН = **рдореБрдирд┐рд░рдпрдореН** (iс╕е + a = ir)
    - рдЧреБрд░реБрдГ + рдмреНрд░рд╣реНрдорд╛ = **рдЧреБрд░реБрд░реНрдмреНрд░рд╣реНрдорд╛**

### 3.4 рд╡рд┐рд╕рд░реНрдЧ рд▓реЛрдк (рд░реЛ рд░рд┐ / рдвреЛ рдвреЗ рд▓реЛрдкрдГ)
- **Rule:** `рд░реН` + `рд░реН` = First `r` is deleted, and preceding vowel is lengthened.
- **Example:**
    - рдкреБрдирд░реН + рд░рдорддреЗ = **рдкреБрдирд╛рд░рдорддреЗ**
